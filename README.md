ReadStart is a web application designed to help people who want to develop a reading habit but struggle to find the right books, stay motivated, and keep track of their progress. Built with a clean, distraction-free design, it allows users to discover books, track reading progress, and visualize their journey through a personal dashboard.

## ğŸ¯ Overview

ReadStart solves the common pain points of building a reading habit:
- **Book Discovery** - Find the perfect book using dual-API search
- **Progress Tracking** - Never lose your place with page bookmarking
- **Motivation** - Visualize your journey with stats and goals
- **Organization** - Keep your reading organized with smart shelves

## âœ¨ Features

### ğŸ“– Core Functionality

- **Dual-API Book Search** - Real-time search by title or author using Open Library and Google Books API
- **Book Detail View** - Dedicated pages showing comprehensive book information
- **Personal Reading Shelves** - Organize books into "Want to Read", "Currently Reading", and "Finished"
- **Page Bookmarking** - Track progress with auto-calculated completion percentages
- **Author Discovery** - Explore an author's complete works and see which books are in your shelves
- **Timeline Filtering** - Browse completed books by Year and Month
- **Personal Reading Dashboard** - View statistics and track progress toward yearly goals
- **Reading Notes** - Save personal thoughts and quotes for each book
